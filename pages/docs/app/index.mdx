App Router 是使用 React 最新特性来构建应用程序的新范式。如果你已经熟悉 Next.js 你会发现 App Router 是现有基于文件系统的路由在 [Pages Router](/docs/pages) 中的自然演进。

对于新的应用程序，我们推荐使用 App Router。对于现有应用程序，你可以 [渐进地迁移到 App Router](/docs/app/building-your-application/upgrading/app-router-migration)。

在 Next.js 13 之前，Pages Router 是在 Next.js 中创建路由的主要方式。它使用文件系统路由器将每个文件映射到路由。Pages Router 在较新版本的 Next.js 中仍然支持，但我们建议迁移到新的 [App Router](/docs/app) 以使用 React 的最新特性。

对于使用 Pages Router 的现有应用程序，请使用文档的此部分。
